Import Export
  

Export

To SQL - When exporting directly to a database, the system will create a table named `db_<FileName>` which is then populated using the MySQL Bulk Copy which is faster than importing a SQL file.

To SQL File - This will generate a MySQL formatted file

To CSV - This will produce a comma seperated file, with escaped strings

To MPQ - This will export the selected file to an MPQ archive, if an existing archive is selected then the file will be appended to it otherwise a new archive will be created

Import

From SQL - This will import data directly from a MySQL database

From CSV

Import Notes:

  • Columns currently need to be identical to the definition file i.e. same column names, types and count
  • Importing data has the choice of:
    1. Importing new rows only
    2. Updating different rows and importing new rows
    3. Overwriting all existing data
