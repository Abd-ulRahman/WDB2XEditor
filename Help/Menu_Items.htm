WDBX Editor Help
Menu Items
  

Below are the menu items located in the top nav bar.

File Menu
  • Open - This allows the choice of multiple of any of the available items
  • Open from MPQ - Will extract the DBC and DB2 files from a designated MPQ archive
  • Open from CASC - Selecting a WoW client directory will allow the ability to extract any DBC and DB2 files
  • Save - WIll save the currently in view file
  • Save All - Will save all loaded files to a specified fodler
  • Reload - Will reload the current file from the base file removing all changes that have not been saved
  • Close - Will close the current file and remove it from the editor
  • Close All - Will close all files and remove them from the editor
Note: Legion ADB files require the DB2 counterpart to be loaded first as the DB2 contains key information for parsing the ADB. The program will prioritise DB2 over ADB if both are loaded together.

Edit Menu

  • Undo
  • Redo
  • Go To - Will go to the row with the input ID number
  • Find and Replace
Note: Undo and Redo history is lost each time a new file is edited

Export

  • To SQL - Will create and insert all data into a MySQL database
  • To SQL File - Generates a MySQL .sql file
  • To CSV
  • To MPQ - Will either generate a new MPQ file or append to an existing one
  • To JSON

Import

  • From SQL
  • From CSV

Tools

  • Definition Editor - Allows the ability to create, edit and delete definitions
  • WotLK Item Import - Fixes the red question mark issue when creating custom items in the database for WotLK
  • WDB5 Parser - Attempts to creates definitions for Legion DB2 files

Context Menus

  • File List View - Has the option to open and close specific files
  • Row Header - Has the option to Go To and to copy and paste a row's data
